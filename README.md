# mvvm

*一个 mvvm 数据双向绑定框架*

*Front-end framework using MVVM*

## How to run the demo
1. install the webpack globally.

```bash
npm install webpack -g
```

2. install the dependencies.
```bash
npm install
```

3. build the source code.
```bash
npm run dev
```

4. check the demo.
* `dist/index.html`
